css选择器

  • 通配符选择器：*
  • 标签选择器：p / div / ul...
  • id选择器
  • 类选择器
  • 后代选择器(所有子孙)：ul li
  • 并集选择器：p,div
  • 子元素选择器(仅到孩子节点)：div > p
  • 属性选择器：span[class], span[class = "ff"], span[class~="ff"], span[class^="f"], span[class$="f"], span[class*="f"]
  • 伪类选择器：:hover, :link, :active, :visited, :focus
  • 伪元素选择器：::before , ::after, :first-child...
